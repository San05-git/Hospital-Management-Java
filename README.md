# Hospital Management System (Java)

A professional Command-Line Interface (CLI) application built during my 6th semester to demonstrate 
efficient data management using the **Queue Data Structure**.

## Key Features
* **FIFO Logic:** Implements First-In-First-Out processing for patient appointments.
* **Encapsulation:** Clean separation of Patient data and Management logic.
* **Professional UI:** Uses ANSI color coding and structured ASCII borders for a clear terminal experience.

## Tech Stack
* **Language:** Java (OpenJDK 25)
* **Data Structure:** LinkedList (Queue Implementation)
* **Environment:** VS Code

## How to Run
1. Compile: `javac Main.java HospitalManager.java`
2. Run: `java Main`
